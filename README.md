# Solución de la ecuación del calor en tres dimensiones

Este repositorio contiene una **solución analítica de la ecuación del calor en tres dimensiones**, desarrollada de manera independiente como material de apoyo y referencia.

El documento principal se encuentra en el archivo:

- **calor.pdf**

donde se presenta el desarrollo matemático completo, junto con las hipótesis, condiciones iniciales y resultados relevantes.
